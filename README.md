# Тест "Кто ты из Винкс?"

## 1. Краткое описание

Данный проект представляет собой интерактивный веб-тест, который позволяет пользователям узнать, какая фея из мультсериала "Винкс" им больше всего подходит. Тест состоит из 12 вопросов с выбором ответа.

## 2. Инструкция по установке

Для запуска теста не требуется установка. Достаточно открыть файл `index.html` в любом современном браузере.

## 3. Инструкция по сборке

Проект не требует сборки. Все файлы написаны на чистом HTML, CSS и JavaScript.

## 4. Документация

Структура проекта:
- `index.html` — главная страница
- `style.css` — стили для оформления
- `script.js` — логика теста (вопросы, подсчёт результатов, картинки)

## 5. Инструкция по совместной работе

Над проектом работали 4 человека. Каждая участница добавляла свои вопросы в файл `script.js`. Работа велась через GitHub.
